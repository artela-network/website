Aspect Programming: Enhance DApp Functionality with Native Extensions
Artela Programming is a universal programming model that supports the dynamic creation of native extensions for blockchain at runtime, called Aspects.
· Aspects can seamlessly combine with existing smart contracts to enhance dApp functionality.
· Aspects can be deployed throughout the entire lifecycle of transactions, with more context of transactions accessible than smart contracts.
· Aspects are executed on the WASM virtual machine with fault isolation.
How Aspects Benefit DApp Building
Maximized Functionality
DApps can leverage Aspect Programming to optimize functionality by customizing the entire transaction lifecycle while preserving EVM-equivalence.
Appchain-scale Customization
DApps can achieve appchain-scale customization in a lightweight manner by deploying their own Aspects, as simple as deploying smart contracts.​
Modular DApp Design
​DApp is no longer a monolithic, tightly knit codebase. Promising improvements in flexibility, reusability and extensibility simply by assembling the Lego of Aspects.​
Runtime Dynamic Extensions
Compared to static precompiled solutions, Aspect enables dApps to achieve dynamic and modular upgrades at runtime, while maintaining base layer stability and security.​
